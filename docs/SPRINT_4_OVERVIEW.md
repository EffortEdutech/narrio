# Narrio Sprint 4 Overview

## Goal
Add AI Writer Assist to the existing writer workflow.

## Scope
1. AI continue chapter
2. AI rewrite chapter
3. AI summarize chapter
4. AI suggest chapter title
5. Save AI outputs as proper chapter versions
6. Mock mode when no API key is configured

## Definition of done
A writer can open the editor and use AI actions that create new chapter versions directly.
