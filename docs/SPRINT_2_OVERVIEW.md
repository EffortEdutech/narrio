# Narrio Sprint 2 Overview

## Goal
Turn Narrio from a reader shell into a writer-capable branching story platform.

## Scope
1. Auth
2. Protected writer area
3. Story creation
4. Chapter creation
5. Chapter versioning
6. Version restore
7. Branch creation from existing branch state

## Not in this sprint
- AI rewrite
- AI continue writing
- comments
- feed ranking
- visual graph editor
- collaboration / co-author roles

## Definition of done
A user can sign in, create a story, write chapter versions, restore an older version, and branch the story into a new path.
