# Sprint 6.0 — Baseline Release Notes

## Release name

```text
Narrio Sprint 5 Complete — ForkCraft Reader Loop Baseline
```

## Summary

This baseline confirms that Narrio now has a working ForkCraft reader loop:

```text
Discover story → Explore timelines → Read chapter → Save waypoint → Fork from chapter → See activity
```

## Included features from Sprint 5

- Narrio product language and ForkCraft branding
- Timeline Explorer
- Fork from Chapter flow
- Waypoints / tagged bookmarks
- Activity Feed stub
- First 60 Seconds onboarding
- Visual Experience Refresh
- Marketing landing page direction

## Baseline limitations

The platform is not yet fully launch-ready because Sprint 6 publishing controls are not implemented yet.

Known planned next steps:

- Publish/unpublish story controls
- Publish/unpublish chapter controls
- Timeline visibility controls
- Public story page polish
- Library discovery improvements
- Creator profile polish
