# Narrio Sprint 3 Overview

## Goal
Make Narrio usable as an end-to-end product flow.

## Scope
1. Story publish controls
2. Story visibility controls
3. Branch visibility controls
4. Chapter publish controls
5. Follow story
6. Bookmark chapter
7. Like chapter version
8. Public writer profile
9. Writer bookmarks page

## Not in this sprint
- AI writing assistant
- comments
- notifications
- ranking feed
- co-author collaboration

## Definition of done
A writer can publish a story and chapters, a reader can discover it in the library, follow it, bookmark chapters, and like the current version.
